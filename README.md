# 🐢 海龟交易分析系统

基于经典海龟交易法则（Turtle Trading System）的 A 股趋势跟踪分析工具，提供 Streamlit 交互式可视化看板。

## 功能特性

- **唐奇安通道突破策略** — 价格突破 N 日最高价入场，跌破 M 日最低价出场
- **ATR 头寸管理** — 根据市场波动率动态计算仓位大小和止损位置
- **金字塔加仓** — 每上涨 0.5 倍 ATR 加仓一次，最多加 3 次
- **多股票池支持** — 沪深 300 成分股 / 全市场（60、00 开头）
- **基本面辅助筛选** — 集成 PE、PB、ROE 指标辅助判断
- **交互式 K 线图** — 支持个股详情查看，含唐奇安通道、ATR 止损线、突破标记
- **历史记录管理** — 自动保存分析结果，支持回顾历史信号
- **CSV 导出** — 一键下载完整分析结果

## 快速开始

```bash

# 启动应用
```

## 策略参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 入场通道周期 | 20 日 | 收盘价突破过去 N 日最高价时入场 |
| 出场通道周期 | 10 日 | 收盘价跌破过去 N 日最低价时出场 |
| ATR 计算周期 | 14 日 | 平均真实波幅计算周期 |
| 每笔风险比例 | 1% | 每笔承担风险占总权益比例 |
| 账户总权益 | 1,000,000 元 | 模拟账户资金规模 |

## 海龟交易法则核心规则

1. **入场** — 收盘价突破前一日唐奇安通道上轨（过去 20 日最高价）
2. **出场** — 收盘价跌破唐奇安通道下轨（过去 10 日最低价）或触发 2 倍 ATR 止损
3. **头寸** — `(权益 × 风险比例) / ATR`，向下取整到 100 股
4. **加仓** — 每上涨 0.5 倍 ATR 加仓一次，最多 3 次
